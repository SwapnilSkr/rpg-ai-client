# Splash character portraits

Generated by the backend script (run from `everlore-server/`):

    bun run gen:splash-avatars

Produces `<voice_key>.webp` here — anime portraits downscaled to 320px (~10KB
each). The splash orbit auto-detects them; missing ones fall back to neumorphic
placeholder discs.

- `bun run gen:splash-avatars --force` — regenerate everything
- `bun run gen:splash-avatars tsundere noir` — only specific voices
- `bun run gen:splash-avatars --out=<dir>` — custom output directory
